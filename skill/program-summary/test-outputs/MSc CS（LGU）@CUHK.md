## 课程与项目特色

**必修课程（3 门，9 学分）**：
- Design and Analysis of Computer Algorithms（计算机算法设计与分析）
- Artificial Intelligence（人工智能）
- Operating System（操作系统）

**选修课程（9 门，27 学分，25+ 门可选）**：
- Deep Learning（深度学习）
- Machine Learning（机器学习）
- Natural Language Processing（自然语言处理）
- Computer Vision（计算机视觉）
- Cloud Computing（云计算）
- Parallel Programming（并行程序设计）
- Algorithm Art and Programming Practice
- Computing Capstone I / II（实践课题）
- Internship Training（实习）
- 及其他 AI/系统方向课程

**项目特色**：
- 学院 CS 排名：2025 CS Rankings AI 方向全国第 16、全球第 25（[官网](https://sds.cuhk.edu.cn/en/page/2132)）
- 强调"AI Full-Stack"全栈人工智能教育，与粤港澳大湾区产业深度合作
- 毕业生可获香港中文大学（教育部认证）硕士学位
- 全程英语授课，国际化师资团队（80+ 位全职教授，35%+ 为院士/会士/顶尖科学家）
- 中国大陆学生须额外完成 2 门公民教育课程（Civic Education Courses）（[官网](https://sds.cuhk.edu.cn/en/page/2132)）
