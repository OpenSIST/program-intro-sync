## 课程与项目特色

### 两条学制轨道

| | MIIS-16（标准） | MIIS-21（深度研究） |
|----|----|----|
| 时长 | 16个月 | 21个月 |
| 最少学分 | 84 units（约7门课） | 108 units（约9门课） |
| 方向专攻 | — | Human Language / Language Technology Application / Machine Learning |

### 课程结构（两轨共同要求）

- **Directed Study（导师指导研究）**：前两学期须完成，教师一对一指导（[CSGrad](https://github.com/csms-apply/csgrad/blob/main/docs/S/CMU%20MIIS.md)）
- **Capstone Project（团队毕业设计）**：最后学期完成，团队协作（[官网](https://miis.cs.cmu.edu/learn-us)）
- **暑期实习**：必须完成（有丰富工作经验者可申请豁免）（[官网](https://miis.cs.cmu.edu/learn-us)）
- 约一半传统课程，另一半为项目导向实践（[官网](https://miis.cs.cmu.edu/)）

### 选课灵活性

- 选课优先级较好，可抢到 CMU SCS 热门课如 Distributed Systems、Operating Systems 等（[CSGrad](https://github.com/csms-apply/csgrad/blob/main/docs/S/CMU%20MIIS.md)）
- 与 MSAII、MCDS 学生共享专属学习设施（[CSGrad](https://github.com/csms-apply/csgrad/blob/main/docs/S/CMU%20MIIS.md)）
- 项目由 LTI Professor James Callan 于2013年创立（[官网](https://miis.cs.cmu.edu/)）

## 录取偏好与信号

> ⚠️ 以下为社区经验，仅供参考，不代表官方立场

- MIIS 定位介于 MLT（偏研究）和 MCDS（偏就业）之间，适合希望兼顾研究与工业的申请者（[OpenSIST 社区](https://opensist.tech)）
- 偏好 AI/NLP 背景，有 ACL、EMNLP、ICLR 等顶会论文（哪怕 Findings 档）可显著提升竞争力（[OpenCS](https://github.com/opencsapp/opencsapp.github.io/blob/master/docs/MIIS%40CMU.md)）
- 录取背景多元：接受相当比例非 CS 背景申请者，也有无发表者成功录取（GPA 86/100，无发表）（[OpenCS](https://github.com/opencsapp/opencsapp.github.io/blob/master/docs/MIIS%40CMU.md)）
- 典型录取者：本科 GPA 3.88–3.98，2-3年科研经验，1-2篇论文，有亚马逊、微软等大厂实习（[CSGrad](https://github.com/csms-apply/csgrad/blob/main/docs/S/CMU%20MIIS.md)）
- 录取存在较大随机性，社区描述近似"lottery-like"，不同本科院校间录取方差极大（[OpenCS](https://github.com/opencsapp/opencsapp.github.io/blob/master/docs/MIIS%40CMU.md)）
- Advanced Study 轨道（21个月）可保留博士转换可能性（[OpenCS](https://github.com/opencsapp/opencsapp.github.io/blob/master/docs/MIIS%40CMU.md)）

## 就业情况

- 历史就业率约 **92%**；官网称"大多数毕业生在毕业后六周内获得 offer"（[CSGrad](https://github.com/csms-apply/csgrad/blob/main/docs/S/CMU%20MIIS.md) / [官网](https://miis.cs.cmu.edu/)）
- 毕业生起薪中位数约 **$150,000 USD**（[CSGrad](https://github.com/csms-apply/csgrad/blob/main/docs/S/CMU%20MIIS.md)）
- 主要职位：Software Engineer（SDE）、Machine Learning Engineer（MLE）（[OpenCS](https://github.com/opencsapp/opencsapp.github.io/blob/master/docs/MIIS%40CMU.md) / [CSGrad](https://github.com/csms-apply/csgrad/blob/main/docs/S/CMU%20MIIS.md)）
- MLE 就业率较好，但直接博士转换率较低（[OpenCS](https://github.com/opencsapp/opencsapp.github.io/blob/master/docs/MIIS%40CMU.md)）
- 代表性雇主：Databricks、TikTok、Apple、IBM、Google 等（[CSGrad](https://github.com/csms-apply/csgrad/blob/main/docs/S/CMU%20MIIS.md) / [官网](https://miis.cs.cmu.edu/)）

## 网申注意事项

- **Video Essay 务必认真准备**：官方标注"strongly recommended"，社区认为实际权重较高（[LTI 官网](https://lti.cmu.edu/academics/masters-programs/miis.html)）
- **Early vs Final 截止**：Early Deadline 申请费较低（$80 vs $100），建议早申（[LTI 申请页](https://lti.cmu.edu/apply/index.html)）
- **GRE 不可忽视**：不同于部分 SCS 项目，MIIS 明确要求 GRE，且需在5年内有效（[LTI 官网](https://lti.cmu.edu/academics/masters-programs/miis.html)）
- **选轨建议**：MIIS-16 适合明确就业导向、希望尽快毕业；MIIS-21 适合有更深研究兴趣或希望增加实习机会（[OpenSIST 社区](https://opensist.tech)）
- 网申材料、套磁等注意事项：待补充（[OpenSIST 社区](https://opensist.tech)）
