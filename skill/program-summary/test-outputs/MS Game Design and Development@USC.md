## 申请要求

- **截止日期**：**11月15日**（Fall入学，仅秋季招生）；[官网](https://cinema.usc.edu/admissions/applicationdeadlines.cfm) ⚠️ 请以最新官网为准
- **录取通知**：次年3月中旬；[官网](https://cinema.usc.edu/admissions/applicationdeadlines.cfm)
- **TOEFL**：最低 100分（各单项最低20分）；[官网](https://cinema.usc.edu/admissions/graduate.cfm)
- **IELTS**：最低 7.0；[官网](https://cinema.usc.edu/admissions/graduate.cfm)
- **PTE**：最低 68；[官网](https://cinema.usc.edu/admissions/graduate.cfm)
- **Cambridge English C1 Advanced**：185分以上，各项不低于169；[官网](https://cinema.usc.edu/admissions/graduate.cfm)
- **语言成绩有效期**：申请提交日前24个月内；[官网](https://cinema.usc.edu/admissions/graduate.cfm)
- **Duolingo**：**不接受**（研究生申请）；[官网](https://cinema.usc.edu/admissions/graduate.cfm)
- **GRE**：暂无信息，建议查阅官网
- **推荐信**：**2封**（至少1封学术推荐信，1封职业推荐信）；[官网](https://cinema.usc.edu/admissions/graduate.cfm)
- **学历要求**：已获认可院校本科学位；[官网](https://cinema.usc.edu/admissions/graduate.cfm)
- **作品集（Portfolio）**：必须提交，通过SlideRoom平台上传；[官网](https://cinema.usc.edu/admissions/procedures/interactive/graduateproceduresMS.cfm)
  - Creative Portfolio List（PDF）：列明所有创作作品及自己的角色
  - Creative Work Sample：选一件最优秀的代表作，可浏览时间不超过10分钟
  - Creative Work Sample Description（PDF，最多20页）：说明创作背景和本人贡献
- **个人陈述**：Cinematic Arts Personal Statement；[官网](https://cinema.usc.edu/admissions/procedures/interactive/graduateproceduresMS.cfm)
- **项目问题**：Project Question（不超过2页），描述一段成功和一段困难的合作经历；[官网](https://cinema.usc.edu/admissions/procedures/interactive/graduateproceduresMS.cfm)
- **面试**：可能有远程视频面试（非必须，不所有申请人都会被邀请）；[官网](https://cinema.usc.edu/admissions/procedures/interactive/graduateproceduresMS.cfm)

## 课程与项目特色

- 项目共38学分：18必修学分 + 16选修学分（从多个核心方向中选取） + 4学分毕业设计（Capstone）；[官网](https://cinema.usc.edu/interactive/index.cfm)
- MS课程与MFA项目前三学期基本一致，为业界导向的快节奏密集型培训；[OpenSIST 社区](https://opensist.tech)
- 课程涵盖：游戏设计、游戏开发、制作、美术、动画、音频、用户研究，以及姿态/沉浸式界面、跨媒体设计、互动电影等前沿方向；[USC Games官网](https://games.usc.edu/)
- 每年举办USC Games Expo，展示70余款学生和校友游戏，吸引大量业界嘉宾；[USC Games官网](https://games.usc.edu/)
- 作品可发布至Steam（提供发布支持）；[USC Games官网](https://games.usc.edu/)
- 位于电影学院（SCA）旗下，擅长互动叙事（Interactive Storytelling）；[OpenSIST 社区](https://opensist.tech)
- 地处洛杉矶，毗邻大量3A游戏公司和影视公司，业界资源丰富；[OpenSIST 社区](https://opensist.tech)

## 录取偏好与信号

> ⚠️ 以下内容来自社区经验，仅供参考，不代表官方立场

- 成绩和语言成绩要求不算极高，专业背景不限，**作品集（Portfolio）是录取核心**；[OpenSIST 社区](https://opensist.tech)
- 作品集建议"精而不多"，展示最出色的代表作即可；[OpenSIST 社区](https://opensist.tech)
- 技术能力（programming skills）是加分项；[OpenSIST 社区](https://opensist.tech)
- 擅长叙事性游戏（Narrative Games）且媒介风格新颖独特者有优势；[OpenSIST 社区](https://opensist.tech)
- 需要体现出"与众不同"的独特视角和创作能力；[OpenSIST 社区](https://opensist.tech)
- 有【无面试直接录取】的情况，但极为罕见；[OpenSIST 社区](https://opensist.tech)
- 竞争激烈，未被录取属正常情况；[OpenSIST 社区](https://opensist.tech)

- 常规时间线：11.15 提交 → 次年1月底面试 → 面试后2-3周出offer → 3月中旬统一通知；[OpenSIST 社区](https://opensist.tech)
