## 学费与奖学金

- **学费（非EU/EEA/瑞士学生）**：约 €15,300 / 年（2023年数据，总计约 €30,600 for 2年）；按学期缴纳（[masters.au.dk/tuitionfees](https://masters.au.dk/tuitionfees)，⚠️ 请以最新官网为准）
- **EU/EEA/瑞士学生**：免学费（[官网](https://masters.au.dk/tuitionfees)）
- **奖学金**：丹麦国家奖学金（Danish State Scholarship），面向非EU/EEA学生自动评选，无需单独申请；自然科学院项目仅覆盖学费减免，不含生活费补贴；每个项目通常仅提供1-2个名额（[masters.au.dk/scholarships-and-grants](https://masters.au.dk/scholarships-and-grants)）
- **申请奖学金截止时间**：须在1月15日（夏季入学）或9月15日（冬季入学）前提交申请方可被考虑（[官网](https://masters.au.dk/scholarships-and-grants)）
- **生活成本**：[GlobalCS](https://github.com/Global-CS-application/global-cs-application.github.io/blob/main/docs/Program/T1.5/Aarhus%20CS.md) 描述其生活成本相对可负担，第一年提供住宿协助

## 申请要求

- **截止日期（夏季入学，8/9月开学）**：
  - 非EU/EEA学生：**1月15日**（[官网](https://masters.au.dk/deadlines-and-important-dates)，⚠️ 请以最新官网为准）
  - EU/EEA学生：**3月1日**（[官网](https://masters.au.dk/deadlines-and-important-dates)）
- **截止日期（冬季入学，1/2月开学）**：
  - 非EU/EEA学生：**9月15日**（[官网](https://masters.au.dk/deadlines-and-important-dates)）
  - EU/EEA学生：**10月15日**（[官网](https://masters.au.dk/deadlines-and-important-dates)）
- **TOEFL**：最低 83 分（2026年1月21日之前的成绩）；2026年1月21日后的成绩要求最低 4.5（新评分体系）（[官网](https://masters.au.dk/am-i-qualified-for-admission/english-language-requirements)）
- **IELTS**：最低 6.5，各分项最低 6.0（[官网](https://masters.au.dk/am-i-qualified-for-admission/english-language-requirements)）
- **GRE**：不要求（[OpenSIST 社区](https://opensist.tech)）
- **推荐信**：不要求（欧洲传统不提交推荐信）（[OpenSIST 社区](https://opensist.tech)）
- **个人陈述**：不要求（[OpenSIST 社区](https://opensist.tech)）
- **本科学历要求**：本科学位（至少180 ECTS），需包含：编程类20 ECTS、计算机系统类20 ECTS、理论计算机/HCI类20 ECTS、数学/统计类20 ECTS（[官网](https://masters.au.dk/computerscience)）
- **课程描述材料**：需提交课程大纲（syllabi），可自行翻译为英文（需保留原文）并请学校教务盖章（[OpenSIST 社区](https://opensist.tech)）
- **申请费**：€150（最多申请同校3个项目）（[官网](https://masters.au.dk/tuitionfees)）

## 课程与项目特色

四个专业方向可选（[官网](https://masters.au.dk/computerscience)）：

1. **Cyber Security（网络安全）**
2. **Artificial Intelligence（人工智能）**
3. **Human-Computer Interaction（人机交互）**
4. **Software Foundations and Technology（软件基础与技术）**

- 前三学期选修方向课程，最终学期完成毕业论文（可与企业合作）（[官网](https://masters.au.dk/computerscience)）
- 课程节奏轻松，每周13-15小时课时，有充裕时间实习或科研（[GlobalCS](https://github.com/Global-CS-application/global-cs-application.github.io/blob/main/docs/Program/T1.5/Aarhus%20CS.md)）
- 每门课程10 ECTS学分，采用口试考核（提供备考工作坊）（[OpenSIST 社区](https://opensist.tech) / [GlobalCS](https://github.com/Global-CS-application/global-cs-application.github.io/blob/main/docs/Program/T1.5/Aarhus%20CS.md)）
- 强势研究方向：**密码学、形式化验证、程序分析、程序语言理论**（[OpenSIST 社区](https://opensist.tech)）
- 国际排名前100（QS或类似体系）（[官网](https://cs.au.dk/education/master/masters-degree-in-computer-science-for-international-students)）
- 大量国际交换机会（[OpenSIST 社区](https://opensist.tech)）
- 第二学期起可加入导师课题组，并获专属办公空间（[GlobalCS](https://github.com/Global-CS-application/global-cs-application.github.io/blob/main/docs/Program/T1.5/Aarhus%20CS.md)）

## 录取偏好与信号

> ⚠️ 社区经验，仅供参考

- 录取核心标准是课程修读情况（是否修读了对方院校本科培养方案中的对应课程），而非软性背景（[OpenSIST 社区](https://opensist.tech)）
- 在欧洲（尤其丹麦）升学传统中，本科生升硕士通常无竞争性选拔；丹麦本科毕业生有法定权利（legal right）进入同专业硕士项目；对国际生申请材料要求因此相对简单（[OpenSIST 社区](https://opensist.tech)）
- 项目录取强调"课程对齐"（program match），而非软性材料（[GlobalCS](https://github.com/Global-CS-application/global-cs-application.github.io/blob/main/docs/Program/T1.5/Aarhus%20CS.md)）
- 中国申请人需注意：丹麦学校对中国院校的评估方式与国内排名体系有所不同，建议提前确认所在院校的认可情况（[GlobalCS](https://github.com/Global-CS-application/global-cs-application.github.io/blob/main/docs/Program/T1.5/Aarhus%20CS.md)）
- 接受录取的时间窗口很短，通常需在一周内决定（[OpenSIST 社区](https://opensist.tech)）

## 就业情况

- 学校提供职业支持和就业安置服务（[官网](https://cs.au.dk/education/master/masters-degree-in-computer-science-for-international-students)）
- **PhD 通道**：硕士第二年可提前加入 PhD 项目（欧洲博士生为定期合同雇员，同时是学生和学校雇员）；每年提供70+博士名额（[OpenSIST 社区](https://opensist.tech) / [GlobalCS](https://github.com/Global-CS-application/global-cs-application.github.io/blob/main/docs/Program/T1.5/Aarhus%20CS.md)）

## 网申注意事项

- 提交申请时需附课程大纲（syllabi），可自行翻译为英文（须保留原文），并请所在学校教务处盖章（[OpenSIST 社区](https://opensist.tech)）
- 申请费 €150，最多同时申请本校3个项目（[官网](https://masters.au.dk/tuitionfees)）
- 语言成绩须在申请门户开放时起有效期不超过两年；学校直接向语言考试机构核实成绩（[官网](https://masters.au.dk/am-i-qualified-for-admission/english-language-requirements)）
- 不接受以下语言测试：Duolingo、PTE、Pearson、Oxford Online Placement Test、Trinity ISE III、Euroexam、Michigan Language Assessment（[官网](https://masters.au.dk/am-i-qualified-for-admission/english-language-requirements)）
- 录取通知发出后接受时限极短（通常约一周），需提前做好决策准备（[OpenSIST 社区](https://opensist.tech)）
- 奖学金无需单独申请，自动随录取评审；须在1月15日（夏季）或9月15日（冬季）截止日前提交申请方可被考虑（[官网](https://masters.au.dk/scholarships-and-grants)）
