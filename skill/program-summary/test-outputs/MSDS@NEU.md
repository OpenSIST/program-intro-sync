## 课程与项目特色

**必修核心课程（16 学时）**（[学术目录](https://catalog.northeastern.edu/graduate/university-interdisciplinary-programs/science-data-ms-bos/)）：
- DS 5110: Essentials of Data Science（4 学时）
- CS 5800 或 EECE 7205: Algorithms（4 学时）
- CS 6140 或 EECE 5644: Machine Learning（4 学时）
- DS 5500: Data Science Capstone（4 学时，必修毕业课）

**方向选修课程（16 学时）**，三大方向：
1. **Computer Science**：涵盖云计算、分布式系统、数据库
2. **Engineering Theory and Modeling**：聚焦能源、工程应用、基础设施
3. **Data, Design, and Visualization**：数据叙事、用户体验、可视化设计

**项目特点**：
- 跨学院跨学科培养（Khoury 计算机学院 + 工程学院，部分方向含设计学院）
- 滚动录取（Rolling admissions）
- 设有 Align 入门路径，适合跨专业申请者
- 毕业生就业方向：数据科学家、机器学习工程师、数据工程师、商业智能开发者等

## 就业情况

根据官网说明，毕业生可从事以下岗位：数据科学家（data scientist）、机器学习工程师（ML engineer）、数据工程师、数据架构师、商业智能开发者、数据分析师等。官网引用数据显示，88% 的数据科学家拥有至少硕士学位。（[官网](https://graduate.northeastern.edu/programs/ms-data-science/master-of-science-in-data-science-boston/)）
