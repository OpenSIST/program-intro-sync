## 申请要求

- **截止日期**：官网 MSAI 页面未直接列明截止日期（⚠️ 请查阅 [官网入学指南](https://www.ntu.edu.sg/admissions/graduate/cwadmissionguide) 或联系 [email protected] 确认）；申请后最长 4 个月出结果
- **TOEFL / IELTS**：TOEFL iBT ≥ 100 / IELTS ≥ 6.5；或提供英语授课证明（MOI）（[官网](https://www.ntu.edu.sg/education/graduate-programme/master-of-science-in-artificial-intelligence)）
- **GRE**：可选（Optional），无强制要求（[官网](https://www.ntu.edu.sg/education/graduate-programme/master-of-science-in-artificial-intelligence)）
- **推荐信**：可选（Optional）；申请系统内申请人自行上传，无需推荐人直接通过系统提交（[官网](https://www.ntu.edu.sg/education/graduate-programme/master-of-science-in-artificial-intelligence)；[GlobalCS](https://github.com/Global-CS-application/global-cs-application.github.io/blob/main/docs/Program/T1.5/NTU%20MSAI.md)）
- **学历要求**：计算机科学、计算机工程或相关领域荣誉学士学位（具备扎实数学基础）；或相关专业本科 + 2 年工作经验（[官网](https://www.ntu.edu.sg/education/graduate-programme/master-of-science-in-artificial-intelligence)）
- **申请材料**：身份证明、简历（CV）、英语水平证明、英文版学位证书及成绩单、个人陈述（SOP）（[官网](https://www.ntu.edu.sg/education/graduate-programme/master-of-science-in-artificial-intelligence)）

## 课程与项目特色

**学分结构（共 30 AUs）**（[官网](https://www.ntu.edu.sg/education/graduate-programme/master-of-science-in-artificial-intelligence)）：
- 核心课程：12 AUs（约 4 门课）
- 选修课程 + Master Project（可选）：18 AUs
- 每门课 3 AUs，13 周共 39 学时
- 全日制学生每学期最多选 5 门课（15 AUs）；兼读制学生每学期最多 3 门课（9 AUs）（[CCDS MSc 项目页](https://www.ntu.edu.sg/computing/admissions/graduate-programmes/master-of-science-programmes)）

**主要研究方向**：计算机视觉、自然语言处理、机器人学、IoT 数据、生成式 AI

**其他特色**：
- Master Project 为可选科研项目（6 AU，历时约一年），可全以选修课替代（[官网](https://www.ntu.edu.sg/education/graduate-programme/master-of-science-in-artificial-intelligence)）
- 完成 MSAI 后可申请 NTU PhD 项目（[GlobalCS](https://github.com/Global-CS-application/global-cs-application.github.io/blob/main/docs/Program/T1.5/NTU%20MSAI.md)）
- 提供 Python Programming 零学分桥接课（选修，适合编程背景较弱者）（[官网](https://www.ntu.edu.sg/education/graduate-programme/master-of-science-in-artificial-intelligence)）
