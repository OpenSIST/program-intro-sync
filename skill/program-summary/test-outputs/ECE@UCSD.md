## 课程与项目特色

项目提供多个专业方向（2025–26），涵盖（[官网](https://ece.ucsd.edu/course-2025-2026)）：

- **信号处理与通信**：数字信号处理（DSP I/II）、数字通信、信息论与编码
- **机器学习与数据科学**：Statistical Learning I/II、深度学习与应用、最优化与计算数据分析
- **VLSI 与电路设计**：VLSI 数字系统算法与架构、VLSI 集成电路与系统设计、模拟电路设计
- **光子学与光学**：激光、光调制、光纤通信、生物医学成像、纳米光子学
- **生物医学工程**：医疗器械、生物医学集成电路、生物物理与成像原理
- **电力系统与能源**：电力电子、能量转换、智能电网、可再生能源

2025–26 新增课程包括：
- ECE 285：Agentic AI and LLM for Smart Grids
- ECE 284：Low-power VLSI Implementation for Machine Learning

**学分要求**（[官网](https://www.ece.ucsd.edu/graduate/ms-degree-requirements)）：
- Plan I（论文方向）：共 52 学分，须完成毕业论文并口头答辩；
- Plan II（综合考试方向）：共 48 学分，须通过 3 门课程综合考试（最多 5 次机会）；
- 两种方案均要求至少 12–16 学分为 200 级以上 ECE 课程；
- 最低在读 3 个学期（Residency Requirement）。
