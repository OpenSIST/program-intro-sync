## 课程与项目特色

项目总计 60 学分，课程由三大模块交叉构成：

**Jacobs Programmatic Core（17 学分）**
- Studio 课程（8 学分）：Product Studio + Startup/BigCo/PiTech Impact Studio 之一 + Studio 选修
- 备课课程（1 学分）
- 两学期 Specialization Project（8 学分）

**Jacobs Technical Core（10 学分）**
- 算法/数据结构或生产技术课（3 学分）
- 机器学习课程（3 学分）
- HCI & Design（3 学分）
- 伦理课（≥1 学分）

**Health Tech Concentration Core（9 学分）**
- Healthcare Organizations & Delivery + 独立研究（3 学分）
- Health Tech Oriented Machine Learning（3 学分）
- Weill Cornell Medicine 认可课程（3 学分）

**选修课（24 学分）**：Health Tech 方向选修（12 学分）+ 通识选修（12 学分，含 CS/ECE/ORIE/INFO/LAW/TECH）

Studio 课程强调跨学科团队合作，学生将与 MBA、法律、工程专业同学共同完成创业或企业级项目，并与 Weill Cornell Medicine 的医生合作。

（[官网课程页](https://studentaffairs.tech.cornell.edu/academics/curriculum/ms-in-information-systems-health-tech/)）
