## 学费与奖学金

- **学费（2025-26学年）**：$6,384 / 门课；12门课合计约 **$76,608**；18门课合计约 **$114,912**。学费预计每财年上涨 3-7%。（[官网 Tuition](https://datascience.uchicago.edu/education/tuition-fees-aid/)，2025 年数据）
- **报名押金**：$1,500（不可退还，计入第一学季学费）（[官网 Tuition](https://datascience.uchicago.edu/education/tuition-fees-aid/)）
- **申请费**：$90（不可退还）（[申请页](https://datascience.uchicago.edu/education/masters-programs/ms-in-applied-data-science/how-to-apply/)）
- **免费非学分课程**：Career Seminar（必修，满3年工作经验可申请豁免）、R/Python 基础课、统计概念导论、高级线性代数（均无额外费用）（[官网 Tuition](https://datascience.uchicago.edu/education/tuition-fees-aid/)）
- **奖学金**：提供 merit-based 部分学费奖学金，申请时自动纳入评审，无需单独申请；建议在早期截止日期前提交以提高获奖概率。（[官网 Tuition](https://datascience.uchicago.edu/education/tuition-fees-aid/)）
- **其他资助**：联邦学生贷款、私人贷款，国际学生亦有对应资源。（[官网 Tuition](https://datascience.uchicago.edu/education/tuition-fees-aid/)）

## 申请要求

- **截止日期（2026年秋季入学）**：
  - 2026-01-26 — 需签证担保的国际申请者优先截止（线下项目）
  - 2026-03-04 — 第二轮截止
  - 2026-05-06 — 第三轮截止
  - 2026-06-23 — 最终截止（线下 & 线上项目）
  - ⚠️ 招满即止，请以最新官网为准：[申请页](https://datascience.uchicago.edu/how-to-apply/)
- **TOEFL**：最低 **104** 分（iBT）（[FAQ](https://datascience.uchicago.edu/education/masters-programs/ms-in-applied-data-science/faqs/)）；部分情况可豁免
- **IELTS**：最低 **7.0**，无子分要求（[FAQ](https://datascience.uchicago.edu/education/masters-programs/ms-in-applied-data-science/faqs/)）
- **GRE**：**不要求**（GRE/GMAT 均为可选，非必须）（[FAQ](https://datascience.uchicago.edu/education/masters-programs/ms-in-applied-data-science/faqs/)）
- **推荐信**：**2封**，至少1封来自直属上级 / 实习导师，证明职场表现；不接受家人、朋友或同龄同学（[申请页](https://datascience.uchicago.edu/how-to-apply/)）
- **其他必要材料**：
  - Candidate Statement（700字以内）
  - 简历 / CV（突出工作经历）
  - 编程作品集 Programming Supplement（不超过2页 PDF，展示 R 或 Python 能力）
  - 视频作品集 Virtual Portfolio（1个必选 + 1个可选，每段约1.5分钟）
  - 所有就读院校的非官方成绩单
  - $90 申请费
  - （[申请页](https://datascience.uchicago.edu/how-to-apply/)）
- **建议申请人背景**：竞争力较强的申请者通常有至少1年全职工作经验及相关实习经历（[官网 In-Person Program](https://datascience.uchicago.edu/education/masters-programs/in-person-program/)）

## 课程与项目特色

**1年制标准 Track（12门课）**（[官网 In-Person Program](https://datascience.uchicago.edu/education/masters-programs/in-person-program/)）：

| 类别 | 门数 | 内容举例 |
|------|------|----------|
| 核心课 | 6门 | Time Series Analysis and Forecasting、Statistical Models for Data Science、Machine Learning I & II、Data Engineering Platforms 或 Big Data/Cloud Computing、Leadership and Consulting for Data Science |
| 选修课 | 4门 | 医疗健康分析、计算机视觉、自然语言处理、强化学习、优化算法、供应链管理、量化金融等 |
| Capstone 项目 | 2门 | 跨2个 quarter，与真实企业合作解决实际业务问题 |

**2年 Thesis Track（18门课，2026年秋季新增，约50-55个名额）**（[FAQ](https://datascience.uchicago.edu/education/masters-programs/ms-in-applied-data-science/faqs/)）：
- 6门核心 + 8门选修 + 2门 Capstone + 2门 Independent Study（论文研究）
- 需完成书面论文或产出 MVP 的 thesis project，由导师监督
- 仅限线下全职学生，申请截止2025-12-04（2026秋季入学已关闭）

**项目亮点**：
- 课程时间友好：线下课以周一至周四 18:00-21:00 为主，兼顾在职人士
- 线上项目设年度 Immersion Weekend 线下活动
- 强调将统计建模、机器学习、数据工程、云计算与 AI 应用于真实业界问题
- 与 Booth 商学院合作提供 MBA/MS 联合学位选项
- 与 MSDS（偏科研/博士路径）不同，MS-ADS 面向有业界经验、希望直接就业的申请人（[官网对比文章](https://datascience.uchicago.edu/news/should-you-choose-ms-in-data-science-or-ms-in-applied-data-science-at-uchicago/)）

## 就业情况

项目定位为培养「job-ready」数据科学从业者，覆盖统计建模、机器学习、数据工程、云计算及 AI 等方向，提供全程职业顾问、网络活动、企业资源对接及实习支持。线下项目具备 STEM/OPT 资格，国际学生 OPT 可延至3年。具体就业率及就业去向数据暂无官网公开，建议联系项目办公室或查阅在校生经验。（[官网 In-Person Program](https://datascience.uchicago.edu/education/masters-programs/in-person-program/)）

## 网申注意事项

- 编程作品集（Programming Supplement）须以 R 或 Python 展示实际编码能力，建议提前准备（[申请页](https://datascience.uchicago.edu/how-to-apply/)）
- 视频作品集（Virtual Portfolio）每段约1.5分钟，建议在安静、专业环境中录制
- 奖学金无需单独申请，申请时自动评审，建议早轮提交（[官网 Tuition](https://datascience.uchicago.edu/education/tuition-fees-aid/)）
- 线上项目不支持 F-1 签证及 STEM OPT，国际学生须选择线下项目（[FAQ](https://datascience.uchicago.edu/education/masters-programs/ms-in-applied-data-science/faqs/)）
- 需签证担保的国际学生须在 2026-01-26 前申请（[申请页](https://datascience.uchicago.edu/how-to-apply/)）
- 推荐人须通过系统发送的链接上传推荐信，注意提前提醒推荐人（[FAQ](https://datascience.uchicago.edu/education/masters-programs/ms-in-applied-data-science/faqs/)）
- 申请平台：[apply-psd.uchicago.edu](https://apply-psd.uchicago.edu/apply/)
