## 学费与奖学金

- **学费**：约 $670.47/学分（Wisconsin 居民）或 $1,503.40/学分（非居民）；30 学分总计约 $20,000（居民）~ $45,000（非居民）。CSGrad 社区数据显示全职 full-time 约 $13,000/学期，总花费约 $35,000（[官网学费页](https://pdc.wisc.edu/degrees/computer-sciences-professional-program-ms/)，2025 年数据，⚠️ 请以最新官网为准）
- **奖学金与资助**：该项目为 self-funded，系里**不提供**直接经费支持，也不保证助教/助研职位，且助教薪资不包含学费减免。可申请批改作业（grader）岗位，时薪约 $22.66（2024-25 学年）（[官网资助页](https://www.cs.wisc.edu/grad/programs/funding/)）

## 申请要求

- **截止日期**：每年 **2 月 15 日**（仅 Fall 入学，不开放春季入学）（[官网](https://www.cs.wisc.edu/graduate/pmp-admissions/)，⚠️ 请以最新官网为准）
- **TOEFL / IELTS**：国际申请者需提交英语水平证明；官网未列具体分数线，第三方数据显示 TOEFL 最低约 92 分，但请以 Graduate School 官网为准（[官网](https://www.cs.wisc.edu/graduate/pmp-admissions/)）
- **GRE**：**不要求**，可选提交（[官网](https://www.cs.wisc.edu/grad/programs/admissions/)）
- **推荐信**：**3 封**；有工作经历者建议至少含 1 封来自工作主管（[官网](https://www.cs.wisc.edu/graduate/pmp-admissions/)）
- **GPA**：无官方最低要求，但录取具有竞争性（[官网](https://www.cs.wisc.edu/graduate/pmp-admissions/)）
- **工作经验**：官方声明**不强制要求**全职工作经验（[官网](https://www.cs.wisc.edu/graduate/pmp-admissions/)）；但社区反映实际录取非常看重毕业后全职工作经历（见下方「录取偏好」）
- **其他材料**：个人陈述（1~1.5 页）、简历、成绩单；需具备 CS 背景或编程经验，以及数据结构、机器组成原理、微积分等基础课

## 课程与项目特色

- **专业方向**（均为选修）：AI、计算生物学、计算机体系结构、图形学、网络、安全、数据库、人机交互（HCI）、数值分析、优化、编程语言、系统研究、理论 CS（[官网](https://www.cs.wisc.edu/grad/programs/pmp/)）
- PMP 学生**不允许**修 CS 系外课程，除非获批
- 可修最多 6 学分研究课，也可以选择完成论文或项目报告代替部分课程学分
- 设有 **Plus One Pathway（POP）**：UW-Madison CS 在读本科生可提前申请，约 3 个学期内完成硕士学位（[官网](https://www.cs.wisc.edu/grad/programs/pmp/pop/)）
- 实习合作企业历史上包括 PayPal、Amazon、Google、Qualcomm、Roku（[官网](https://www.cs.wisc.edu/grad/programs/pmp/)）
- 社区评价：项目内容类似 GaTech MSCS，学费较低，整体 peer pressure 小（[CSGrad](https://github.com/csms-apply/csgrad/blob/main/docs/A%2B/Wisc%20CS%20PMP.md)）

## 录取偏好与信号

> ⚠️ 以下为社区经验，仅供参考，非官方信息

- 官网称不要求工作经验，但实际录取**极度偏好有毕业后全职工作经历**的申请者。项目曾直接邮件通知申请者：实习经历不算，只有「毕业后的工作」才计入工作经历栏。有 SDE 全职经历者录取率约 85%，无全职经历者约 0.5%（[CSGrad](https://github.com/csms-apply/csgrad/blob/main/docs/A%2B/Wisc%20CS%20PMP.md)）
- 申请人数持续增长：22Fall 约 900 人，23Fall 超过 1000 人；录取人数也在逐年增加（[CSGrad](https://github.com/csms-apply/csgrad/blob/main/docs/A%2B/Wisc%20CS%20PMP.md)）
- 24Fall 录取代表性案例：UCI CS 本科 GPA 3.76；印度本科 + 美硕 + 纽约七年工作经验；IIT 本科 + AWS 两年全职；印度本科 + 三星两年全职（[CSGrad](https://github.com/csms-apply/csgrad/blob/main/docs/A%2B/Wisc%20CS%20PMP.md)）
- 另一个入学通道：先申请在线 **PCP（Professional Capstone Certificate）**，通过 GPA 表现后可申请转入 PMP（[CSGrad](https://github.com/csms-apply/csgrad/blob/main/docs/A%2B/Wisc%20CS%20PMP.md)）
- MSCS（研究型）与 PhD 共用一个 pool，录取更难；PMP 相当于其他学校的 MCS/coursework MS（[CSGrad](https://github.com/csms-apply/csgrad/blob/main/docs/A%2B/Wisc%20CS%20PMP.md)）

## 就业情况

录取同学几乎都有全职工作经历，没有同校 peer pressure，整体求职表现良好。已知就业案例（含实习转正）：

- Amazon（实习 + Return Offer）
- Pinterest（实习 + Return Offer）
- Facebook/Meta（实习 + Return Offer）
- Dell EMC（实习 + Return Offer）
- IIT 背景 + AWS 经历 → 上岸 SF 小厂
- 印度本科 + 三星两年经验 → 上岸 Amazon

（[CSGrad](https://github.com/csms-apply/csgrad/blob/main/docs/A%2B/Wisc%20CS%20PMP.md)）

## 网申注意事项

- 仅接受 **Fall 入学**，每年 2 月 15 日截止，不接受春季申请（[官网](https://www.cs.wisc.edu/graduate/pmp-admissions/)）
- **申请材料中工作经历栏请只填「毕业后全职工作」**，切勿将实习填入工作经历，程序曾专门邮件说明此点（[CSGrad](https://github.com/csms-apply/csgrad/blob/main/docs/A%2B/Wisc%20CS%20PMP.md)）
- PMP 学生不能享受 SuccessWorks 的个性化就业辅导，但可访问 Graduate School 就业发展资源（[官网](https://www.cs.wisc.edu/grad/programs/pmp/)）
- 国际学生可申请 CPT 进行实习，需提前获得授权（[官网](https://www.cs.wisc.edu/grad/programs/pmp/)）
- 推荐信来源：有工作经历者建议含 1 封主管推荐信，但学术推荐仍有价值（[官网](https://www.cs.wisc.edu/graduate/pmp-admissions/)）
- 可考虑先申请在线 PCP 证书项目，后转入 PMP 作为备选路径（[CSGrad](https://github.com/csms-apply/csgrad/blob/main/docs/A%2B/Wisc%20CS%20PMP.md)）
