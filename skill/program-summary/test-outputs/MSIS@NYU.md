## 课程与项目特色

项目结合 Courant 计算机系研究生课程与 Stern MBA 课程，被官方描述为"缩略版 CS 硕士 + MBA"：

**Stern 课程（5 门，15 学分）**
- 2 门通识商业核心（领导力、财务会计等）
- 3 门科技方向课（数字战略、商业数据挖掘、金融信息系统等）

**计算机系课程（5 门，15 学分）**
- 必修：Fundamental Algorithms（CSCI-GA.1170）
- 从 Data Communications & Networks / Operating Systems / Database Systems 中选 2 门
- CS 选修 2 门（含计算金融可作 CS 选修）

**Capstone（1 门）**
- Information Technology Projects（CSCI-GA.3812）：团队项目，与本地机构合作

**选修（2 门）**
- 可从 CS 或 Stern 任选，Stern 课程上限 21 学分

选课优先级高，仅次于 MBA 学生；商学院课程评价普遍优于 CS 课程；workload 不小，但有一些相对轻松的课可选。（[OpenSIST 社区](https://opensist.tech) / [官网](https://cs.nyu.edu/home/master/current_msisreq.html)）
