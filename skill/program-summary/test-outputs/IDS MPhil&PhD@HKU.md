## 学费与奖学金

- **学费（composition fee）**：HK$44,500 / 年（2025/26，全日制）；预计 HK$47,000（2026/27）、HK$49,500（2027/28）。（[HKU Graduate School](https://gradsch.hku.hk/prospective_students/fees_scholarships_and_financial_support/fees)，2025 年数据）
- **基础研究生奖学金（PGS）**：HK$19,135 / 月（2025年9月起）；PhD 试读期确认后提升至 HK$19,655 / 月。奖学金已覆盖学费，实际到手约 18,000 HKD 左右（扣除学费后，社区数据显示约 18K HKD/月，与官方 stipend 基本匹配）。（[Faculty of Science, HKU – Stipend and Support](https://www.scifac.hku.hk/rpg/programme-info/stipend-support)）
- **香港博士研究生奖学金（HKPFS）**：竞争性奖学金，年度 stipend HK$340,800（约 HK$28,400/月）+ 差旅津贴 HK$14,200/年 + 额外现金奖励（Year 1: HK$40,000；其余年份 HK$20,000/年）+ 学费全免，资助期最长3年。（[HKU Graduate School – HKPFS](https://gradsch.hku.hk/prospective_students/fees_scholarships_and_financial_support/hong_kong_phd_fellowship_scheme)）
- **注**：社区提示 stipend 的一部分需用于缴纳学费，PhD 与 MPhil 在 stipend 标准上无区别。（[OpenSIST 社区](https://opensist.tech)）

## 申请要求

- **截止日期**（2025/26 申请季，⚠️ 请以最新官网为准）：
  - Main Round：2025年9月1日 – 2025年12月1日（强烈建议此轮申请）
  - 1st Clearing Round：2025年12月2日 – 2026年4月30日
  - 2nd Clearing Round：2026年5月1日 – 2026年8月31日
  - HKPFS 通常于9月至12月上旬征集申请
  （[官网](https://datascience.hku.hk/study/research-postgraduate/)）
- **学历要求**：
  - PhD：荣誉学士 + 授课型硕士；或第一荣誉（First Class Honours）学士
  - MPhil：荣誉学士（或同等学力）
  （[官网](https://datascience.hku.hk/study/research-postgraduate/)）
- **GPA 要求**：社区数据：本科申 PhD 需 GPA ≥ 3.6（conditional offer 中可能要求毕业时达标，大四需尽量稳住 GPA）。（[OpenSIST 社区](https://opensist.tech)）
- **TOEFL / IELTS**：TOEFL iBT ≥ 80；IELTS ≥ 6.0（各单项 ≥ 5.5）。（[HKU Graduate School – Admission Requirement](https://gradsch.hku.hk/prospective_students/application/admission_requirement)，2026 年起要求成绩在入学年9月1日前两年内取得）
- **GRE**：暂无信息，建议查阅官网
- **推荐信**：暂无信息，建议查阅官网（港校研究型项目通常需2–3封）
- **申请方式**：通过 HKU Graduate School 在线系统申请，选择"Data Science"作为 interdisciplinary area。（[官网](https://datascience.hku.hk/study/research-postgraduate/)）

## 课程与项目特色

- IDS 为 HKU 技术发展蓝图中首批十大研究院之一（HKU Musketeers Foundation Institute of Data Science），处于快速发展期，资源充沛。（[OpenSIST 社区](https://opensist.tech)）
- 研究方向涵盖 CV / NLP / ML / Robotics 等热门领域，导师阵容包括 Prof Yi Ma、Prof Tao Yu、Prof Xihui Liu、Prof Yingyu Liang、Prof Chao Huang 等20余位学者，年轻教授居多，国际发表活跃（顶会 spotlight/oral 常见）。（[官网](https://datascience.hku.hk/study/research-postgraduate/) / [OpenSIST 社区](https://opensist.tech)）
- 课程设置分基础、计算、应用三类模块：
  - MPhil：2个必修模块（共6门）
  - 3年制 PhD：1个必修模块
  - 4年制 PhD：2个必修模块（共6门）
  （[官网](https://datascience.hku.hk/study/research-postgraduate/)）
- IDS 设有**暑研项目**，大二大三同学可申请以提升背景。（[OpenSIST 社区](https://opensist.tech)）
- 与北美学术界联系较强（connection 好），适合以港校 MPhil 为北美 PhD 跳板。（[OpenSIST 社区](https://opensist.tech)）

## 录取偏好与信号

> ⚠️ 以下为社区经验，仅供参考

- 强 Prof 制：录取与否高度依赖与意向导师的匹配度，建议提前联系导师。（[OpenSIST 社区](https://opensist.tech)）
- 本科直申 PhD 有 GPA 3.6 的录取 bar（部分 conditional offer 要求毕业时达标），大四 GPA 尽量保持稳定。（[OpenSIST 社区](https://opensist.tech)）
- 与导师不熟悉时，硬申至少需有1篇顶会论文经历。（[OpenSIST 社区](https://opensist.tech)）
- 一般导师招人均会安排面试。（[OpenSIST 社区](https://opensist.tech)）
- 申请季全年开放，即使北美申请失败，次年春季仍可申请（1st/2nd Clearing Round）。（[OpenSIST 社区](https://opensist.tech) / [官网](https://datascience.hku.hk/study/research-postgraduate/)）
- 港校整体看重高 GPA 和本科院校背景，部分顶尖国内高校（如中科大）学生可能相对吃亏。（[OpenSIST 社区](https://opensist.tech)）

## 就业情况

- 年轻导师发表量大，顶会 spotlight/oral 较多，学术氛围较强，适合有意继续读博或走学术路线的申请者。（[OpenSIST 社区](https://opensist.tech)）
- 具体就业去向数据暂无信息，建议查阅官网或联系在读学生。

## 网申注意事项

- 申请系统中须在 "Interdisciplinary areas" 下选择 "Data Science"，而非选具体院系。（[官网](https://datascience.hku.hk/study/research-postgraduate/)）
- 强烈建议在 Main Round（9月–12月）内提交申请，越早越好。（[官网](https://datascience.hku.hk/study/research-postgraduate/)）
- HKU 附近（薄扶林一带）房租较贵，生活成本需提前规划。（[OpenSIST 社区](https://opensist.tech)）
- PhD 与 MPhil 的 stipend 相同，但部分导师对两者可能有不同对待方式。（[OpenSIST 社区](https://opensist.tech)）
- HKPFS 为竞争性高额奖学金，符合条件者建议同步申请，截止时间通常为每年12月初。（[官网 – HKPFS](https://gradsch.hku.hk/prospective_students/fees_scholarships_and_financial_support/hong_kong_phd_fellowship_scheme)）
