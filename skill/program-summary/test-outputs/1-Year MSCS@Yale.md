## 学费与奖学金

- **学费**：约 $50,900 / 学年（2025–2026 全日制参考数据）（[Yocket](https://yocket.com/universities/yale-university/computer-science-820)，2025–2026 数据）
- **奖学金**：官方明确 MS 一年制项目**无任何财务资助**，学生须自付全部学费。Teaching Fellow 职位偶有空缺，但薪酬不足以覆盖学费。（[官网](https://engineering.yale.edu/computer-scienceacademics/graduate-program/master-science)）

## 申请要求

- **截止日期**：Fall 2025 申请截止 **2025 年 1 月 2 日**（[Yale GSAS FAQ](https://engineering.yale.edu/academic-study/departments/computer-science/graduate-study/graduate-program-admissions-faq)；⚠️ 请以最新官网为准）
- **TOEFL**：总分要求**明显高于 100**；口语单项 iBT 26+ 可豁免大学英语测试。无绝对截止线，但高分更具竞争力。（[官网 FAQ](https://engineering.yale.edu/academic-study/departments/computer-science/graduate-study/graduate-program-admissions-faq)）
- **IELTS**：官网 FAQ 未专门提及，建议查阅 Yale GSAS 官网
- **GRE**：官网表示录取学生通常**达到 90 百分位**（约 328+）；历史上社区反馈严卡 328，22fall 起有录取 GRE325 案例，有所放宽（[官网](https://engineering.yale.edu/computer-scienceacademics/graduate-program/master-science) / [CSGrad](https://github.com/csms-apply/csgrad/blob/main/docs/A/Yale%20MSCS.md) / [OpenCS](https://github.com/opencsapp/opencsapp.github.io/blob/master/docs/One-Year%20MSCS%40Yale.md)）
- **推荐信**：官网强调需要「strong recommendation letters from people who know the applicant」；第三方来源参考为 3 封（[Yocket](https://yocket.com/universities/yale-university/computer-science-820)），建议以官网最新要求为准
- **前置课程**：需有算法、系统编程、计算机体系结构等课程基础（超出基础编程和数据结构）（[官网](https://engineering.yale.edu/computer-scienceacademics/graduate-program/master-science)）
- **其他**：支持 Interfolio 提交推荐信，但 Interfolio **不会填写 Evaluation form**，需推荐人另行完成该表格（[OpenCS](https://github.com/opencsapp/opencsapp.github.io/blob/master/docs/One-Year%20MSCS%40Yale.md)）

## 课程与项目特色

需完成 **8 门**来自认可列表的课程；可用 CPSC692（独立研究，一学期，须获导师及院系批准）替换其中一门。课程与本科生混班，班级规模普遍较小，选课灵活，有难有易。因资源充裕，部分同学会联系导师进组做研究。学校位置较偏，距纽约有一定车程。（[官网](https://engineering.yale.edu/computer-scienceacademics/graduate-program/master-science) / [CSGrad](https://github.com/csms-apply/csgrad/blob/main/docs/A/Yale%20MSCS.md)）

**Defer 机制**：可 defer 一个学期或一年；美本同学可选择 defer 一学期后 spring 入学拿 CPT，一定程度弥补一年制项目实习难题，具体 CPT 批准细节需与学校 DSO 确认。（[CSGrad](https://github.com/csms-apply/csgrad/blob/main/docs/A/Yale%20MSCS.md)）

## 录取偏好与信号

> ⚠️ 社区经验，仅供参考

- 录取几乎只看 GPA、本科学校、TOEFL、GRE，官方明确表示 research is not required，不看实习不看科研。（[CSGrad](https://github.com/csms-apply/csgrad/blob/main/docs/A/Yale%20MSCS.md)）
- 历史上严卡 GRE 328，22fall 起有录取 <328（如 GRE325）的案例，近年有放缓迹象。（[CSGrad](https://github.com/csms-apply/csgrad/blob/main/docs/A/Yale%20MSCS.md) / [OpenCS](https://github.com/opencsapp/opencsapp.github.io/blob/master/docs/One-Year%20MSCS%40Yale.md)）
- 浙大友好，历史上 CS+浙大-UIUC 合办项目学生录取可达 20+ 人。（[CSGrad](https://github.com/csms-apply/csgrad/blob/main/docs/A/Yale%20MSCS.md)）
- 中游 985 各收 1 人左右，录取集中在学校背景 + 三维较强的学生。（[OpenCS](https://github.com/opencsapp/opencsapp.github.io/blob/master/docs/One-Year%20MSCS%40Yale.md)）
- 转专业同学可尝试，但录取难度较高，重视三维。（[OpenCS](https://github.com/opencsapp/opencsapp.github.io/blob/master/docs/One-Year%20MSCS%40Yale.md)）

## 就业情况

一年制项目找工整体偏难，主要因为缺少美国实习经历，24ng 上岸率约 10%–15%，整体不如同为一年制的 UCB EECS MEng。上岸主要靠 TikTok 和 Amazon 捞人。部分 OPT 期间上岸案例：

- 武汉大学 CS + TikTok 国内一年实习 → 上岸 TikTok NG
- 武汉大学软件工程 + 蚂蚁 4 个月实习 → 上岸 2 Sigma
- Tulane Math+CS 无实习 → 上岸小厂
- NYU + 百度实习 → 上岸 AWS NG
- UW CS + AWS intern → 上岸 DoorDash NG
- Waterloo + 加拿大小厂实习 → 上岸 Google NG
- Yale CS + 3 年创业 + 小厂实习 → 上岸 Google NG

（[CSGrad](https://github.com/csms-apply/csgrad/blob/main/docs/A/Yale%20MSCS.md)）
