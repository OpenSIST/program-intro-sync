## 申请要求

- **截止日期**：**2026 年 12 月 9 日（Eastern 时间下午 3 点）**（Fall 2027 入学，[官网 Apply 页](https://mse.s3d.cmu.edu/applicants/apply.html)，⚠️ 请以最新官网为准）
- **TOEFL / IELTS**：TOEFL iBT 最低 100 分（录取均值约 109 分）；IELTS 最低 7.0；Duolingo 最低 120 分。母语非英语且将在美国完成四年本科教育者可申请豁免。（[官网申请页](https://mse.s3d.cmu.edu/applicants/mse-as/apply.html)）
- **GRE**：可选，不提交不会处于劣势；提交者无最低分要求，历史录取均值约 Verbal 159 / Quant 168（[官网 Eligibility 页](https://mse.s3d.cmu.edu/applicants/mse-ap/eligibility.html)，2020 年数据）
- **推荐信**：至少 **3 封**，来自学术或业界人士（[官网申请页](https://mse.s3d.cmu.edu/applicants/mse-as/apply.html)）
- **最低 GPA**：2.5（累积 GPA）
- **背景要求**：CS 或工程相关本科学位；须有离散数学或统计、数据结构与算法、编程语言（Java/C++/Python）基础知识；行业经验不超过 2 年（有 1 年以上实习经历者优先）
- **其他材料**：
  - Statement of Purpose（个人书写，需包含职业目标详述）
  - 视频短片（Video Essay，最长 3 分钟，申请系统中提交）
  - 简历/CV
  - 非官方成绩单（所有就读院校）
  - 在线编程测试（Online Coding Assessment，申请截止后发送邀请，约一周内完成）
  - 申请费：$100
  - 来源：[官网申请页](https://mse.s3d.cmu.edu/applicants/mse-as/apply.html)

## 课程与项目特色

项目时长 16 个月，以 7 周 mini-semester 形式推进，支持同时学习多个方向并逐步递进。（[官网课程计划页](https://mse.s3d.cmu.edu/applicants/mse-ap/plan.html)）

**Semester 1-1（秋季前半）**
- Communications for Software Leaders I
- Statistics for Decision Making
- Product Management Essentials I
- Formal Methods
- Design Patterns

**Semester 1-2（秋季后半）**
- Communications for Software Leaders I（续）
- Agile Methods
- Quality Assurance
- API Design for Scalable Systems
- Requirements for Information Systems

**Semester 2-1（春季前半）**
- Communications for Software Engineers II
- Software Project Management
- Software Architecture
- DevOps: Engineering for Secure Development and Deployment
- 一门选修课

**Semester 2-2（春季后半）**
- Communications for Software Leader II（续）
- DevOps: Engineering for Secure Development and Deployment（续）
- Quality Management
- Engineering Scalable Systems
- 一门选修课

**Semester 3（暑期）**
- Internship for Software Engineers（强制）

**Semester 4（秋季）**
- Software Engineering Practicum（行业赞助毕业实践）+ 一门选修课

**项目特色：**
- 强调大规模、数据密集型和智能系统的持续集成与高速迭代（DevOps 导向）
- Scalable Systems 赛道侧重 DevOps 课程（相对 Embedded Systems 赛道的嵌入式系统课程）
- 所有学生配有 CMU MSE 校友担任职业导师，从第一学期即开始准备实习
- 毕业实践（Practicum）为行业赞助的真实项目
