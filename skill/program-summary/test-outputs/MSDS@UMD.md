## 课程与项目特色

**核心课程（7 门，21 学分）**，来源：[学术目录](https://academiccatalog.umd.edu/graduate/programs/data-science-sads/data-science-ms/)：

| 课程编号 | 课程名称 |
|----------|----------|
| DATA 601 | Probability and Statistics |
| DATA 602 | Principles of Data Science |
| DATA 603 | Principles of Machine Learning |
| DATA 604 | Data Representation and Modeling |
| DATA 605 | Big Data Systems |
| DATA 606 | Algorithms for Data Science |
| DATA 607 | Communication in Data Science and Analytics |

**选修课（3 门，9 学分）**：涵盖深度学习、自然语言处理等高级方向（[官网](https://cmns.umd.edu/graduate/science-academy/data-science/masters)）

**五大能力模块**：Statistics / Machine Learning / Computing / Communication（[官网](https://cmns.umd.edu/graduate/science-academy/data-science/masters)）

**项目特点**：
- 面向职场人士设计，强调实践知识，**不提供科研机会**；有意攻读博士的申请人建议选择研究型项目（[官网](https://cmns.umd.edu/graduate/science-academy/data-science/masters)）
- 以 College Park 校区面授晚间课程为主，同时提供在线选项（[官网](https://cmns.umd.edu/graduate/science-academy/data-science/masters)）
