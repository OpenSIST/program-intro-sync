## 课程与项目特色

项目由 **Kahlert School of Computing** 教师授课，40 学分，所有课程均为 4 学分制。课程安排如下（[官网 Academics](https://msd.utah.edu/academics)）：

**Fall 1**
- CS 6010 Introduction to Software Development（C++、Linux、版本控制）
- CS 6011 Computer Programming（Java、OOP、多线程）
- CS 6012 Data Structures and Algorithms（复杂度分析、图论）

**Spring 1**
- CS 6015 Software Engineering（规格说明、测试、代码质量）
- CS 6013 Systems 1（计算机体系结构与操作系统）
- CS 6014 Systems 2（网络与安全、密码学）

**Summer 1**
- CS 6016 Database Systems and Applications（SQL、NoSQL、MapReduce）
- CS 6017 Data Analytics, Visualization, and Machine Learning

**Fall 2**
- CS 6018 Application System Design and AI Integration
- CS 6019 MSD Capstone Project 或 CS 6020 MSD Capstone Internship（二选一）

**证书选项**（8 个月短期）：
- Programming Certificate：Fall 学期 + 一门 Spring 课程
- Systems & Data Certificate：Spring + Summer 学期

## 网申注意事项

- 项目仅接受 **Fall 学期**入学，无 Spring 入学选项（[官网 FAQ](https://msd.utah.edu/faq)）
- 课程为**全日制线下**，无在线选项，每周须在校至少 20 小时（[官网 FAQ](https://msd.utah.edu/faq)）
- GRE 为可选，但若提交将被纳入评估（[官网 Admissions](https://www.msd.utah.edu/admissions)）
- 申请费：国内 $55，国际 $65（[官网 Admissions](https://www.msd.utah.edu/admissions)）
- 先修课须为本科或更高层次，高中数学不计入（[官网 Admissions](https://www.msd.utah.edu/admissions)）
- 联系方式：msd-advisors@cs.utah.edu / 801-581-7631
