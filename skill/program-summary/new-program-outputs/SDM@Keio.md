## 课程与项目特色

项目为跨学科工程管理方向，英日双语授课，英语学生无需日语即可完成学业（[官网 FAQ](https://www.sdm.keio.ac.jp/en/faq.html)）。

**必修课（共 20 学分）**：
- 核心课（8 学分）：系统设计与管理导论、系统架构与集成、系统验证与确认、项目管理
- 特别研究（8 学分）：以小组形式开展设计项目研究，毕业论文须独立撰写
- 设计项目（4 学分）：围绕可持续社区设计、人机共生系统等宏观主题开展团队实战

**选修课（16 学分以上）**：覆盖技术与社会技能两大方向，可选修其他庆应大学院课程（学分不计入毕业要求）

毕业总学分要求：36 学分，须通过硕士论文及终试（[官网](https://www.sdm.keio.ac.jp/en/education/degree/master.html) / [课程页](https://www.sdm.keio.ac.jp/en/education/curriculum.html)）

另设在职生专项通道（Learning-Intensive Course），周末及晚间开课，适合在职人士（[官网](https://www.sdm.keio.ac.jp/en/admission/)）。

- 项目**主要推荐转码/转方向同学申请**，对 CS 背景申请者吸引力相对有限（[GlobalCS](https://github.com/Global-CS-application/global-cs-application.github.io/blob/main/docs/Program/TX/Keio%20SDM.md)）

## 就业情况

- 在日本就业市场认可度高，特别适合希望留日发展的申请者（[GlobalCS](https://github.com/Global-CS-application/global-cs-application.github.io/blob/main/docs/Program/TX/Keio%20SDM.md)）
- 项目为跨学科方向，非纯 CS 学位，对赴第三国就业或进入顶尖科技公司的帮助有限（社区来源）
- 具体就业数据暂无信息，建议查阅官网

## 网申注意事项

- 须先通过 [Pre-Contact 表格](https://wwwdc01.adst.keio.ac.jp/kj/sdm/precontact-en.html) 联系目标导师并获认可，之后方可正式提交申请（[官网 FAQ](https://www.sdm.keio.ac.jp/en/faq.html)）
- 正式申请须先完成线上 Web Entry System 注册，再将纸质材料邮寄至学务办（[官网](https://www.sdm.keio.ac.jp/en/admission/guidelines.html)）
- 国内邮寄以邮戳日期为准；从海外寄出须在截止日期前到达（[官网](https://www.sdm.keio.ac.jp/en/admission/guidelines.html)）
- 若无法满足本科学历要求（15 年学制），需提前申请资格预审（[官网](https://www.sdm.keio.ac.jp/en/faq.html)）
- 学费较高（约 200 万日元/年），为日本私立大学中偏高水平（社区来源）
