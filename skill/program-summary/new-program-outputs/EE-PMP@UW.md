## 课程与项目特色

项目共需 45 学分，含 36–40 学分课程 + 5–9 学分 Colloquium 研讨课（Credit/No-Credit，远程提交读书笔记形式）。新生推荐首 quarter 修 EE P 502（Analytical Methods for Electrical Engineering，4 学分）。

课程覆盖方向广泛，可选全软件方向课程，代表性课程包括（[社区来源](https://github.com/csms-apply/csgrad/blob/main/docs/A/uw%20ee%20pmp.md)）：
- Data Structures and Algorithms for ECE
- Deep Learning for Big Visual Data
- Computer Vision Methods
- Large Language Models / ChatGPT
- TinyML for Edge Computing

项目为晚间面授，兼顾在职人士，不设论文要求（可选研究方向）。UW ECE 研究生项目在 U.S. News 2025–26 排名全美第 18 位。
