## 课程与项目特色

CS37 项目专为本科非 CS 背景（工程、理科等）的学生设计，课程结构分三层：

- **预备课程（4 学分）**：CSCI 455 编程系统设计导论
- **基础课程（8+ 学分）**：EE 457 或 EE 450（计算机系统）、CSCI 402 操作系统
- **选修课（500–600 级）**：涵盖 AI、Web 技术、多媒体系统、软件工程、数据库等

预备和基础课程**必须在 USC 完成，不可转课或豁免**。项目不含论文选项，但第一学期后可申请 directed research。可选在线就读（DEN@Viterbi），学费相同。具备 OPT STEM 延期资格。（[官网](https://viterbigradadmission.usc.edu/programs/masters/msprograms/computer-science/ms-cs-scientists-engineers/)）
