## 课程与项目特色

项目设有 **3 门核心课程**，需分别从算法、智能系统、计算机系统三个方向各选一门，外加集中方向选修课，通过以下三条轨道之一完成学位（[官网](https://computerscience.emory.edu/graduate-ms/cs-ms-programs.html)）：

- **Project Track**：5 门选修课 + 6 学分项目研究（CS 597R/599R）
- **Coursework Track**：7 门选修课（纯课程）
- **Thesis Track**：4 门选修课 + 9 学分论文研究（CS 599R）

**核心课程示例**（[官网](https://computerscience.emory.edu/graduate-ms/cs-ms-programs.html)）：
- 算法方向：CS 523 数据结构与算法 / CS 526 算法
- 智能系统方向：CS 534 机器学习 / CS 557 人工智能 / CS 570 数据挖掘
- 计算机系统方向：CS 551 系统编程 / CS 554 数据库系统 / CS 580 操作系统

研究方向涵盖 AI、ML、NLP、数据挖掘、生物/医疗信息学、高性能计算、理论与系统（[官网](https://computerscience.emory.edu/graduate-ms/index.html)）。

所有学生须完成一项 **Practicum**（重要项目或实习），与教师或行业专家合作（[官网](https://computerscience.emory.edu/graduate-ms/cs-ms-programs.html)）。

社区来源提到，Emory CS 与 BMI（生物医学信息学）双轨道设置，项目侧重医疗健康领域的 ML/AI 研究，师生比较低，科研机会多（[CSGrad](https://github.com/csms-apply/csgrad/blob/main/docs/B/Emory%20MSCS.md)）。
