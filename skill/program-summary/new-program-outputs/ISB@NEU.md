## 课程与项目特色

该项目以"商业管理、软件工程与信息技术"交叉为核心，目标是培养 IT 领导者。项目提供多种专注方向（Focus Areas），包括：AI Systems Engineering、Business Intelligence and Data Analytics、Full-Stack Software Engineering、Machine Learning and Data Sciences Engineering、Project/Product Management、User Experience Design，以及 Medical Software Engineering 专项方向。

毕业选项分三种：纯课程（Coursework）、项目（Project）、论文（Thesis），共需完成 32 学分。学生可在 INFO、CSYE、DAMG、TELE 等多个学科的课程中自由选修。（[官网](https://catalog.northeastern.edu/graduate/engineering/multidisciplinary/information-systems-msis/)）

Co-op 计划最长 8 个月，合作雇主超过 3,000 家，包括 Google、Amazon、Goldman Sachs、Tesla、Apple 等。（[官网](https://coe.northeastern.edu/academics-experiential-learning/academic-departments/seis/academics/ms-insy/)）
