## 课程与项目特色

本项目要求至少 **30 学分**，构成如下（[官网](https://engineering.tamu.edu/cse/academics/degrees/graduate/meng.html)）：

- **CSCE 研究生课程**：12+ 学分（不含 CSCE 681/684/685/691）
- **ECEN 研究生课程**（非 CSCE 交叉课）：6+ 学分 —— 这是与 MCS 的核心区别，需要额外 ECE 方向课程
- **选修研究生课程**（CSCE、ECEN 或批准课程）：6+ 学分
- **CSCE 681**（研究生研讨会）：1 学分
- **400 级本科课程**：最多 1 门（3 学分，需导师批准）
- **CSCE 685**（定向研究）：最多 3 学分

**与 MCS 的关系**：CE MEng 与 MCS 高度相似，实质差别在于需额外完成 6 学分 ECEN 课程。CSE 系内部支持 MCS/MSCE/MSCS/MEng 等项目间转换（社区来源 OpenCS）。

**在线授课**：本项目同时提供异步在线版本（[官网](https://engineering.tamu.edu/cse/academics/degrees/graduate/online-degrees/master-computer-engineering.html)）。

## 录取偏好与信号

> ⚠️ 社区经验，仅供参考

- GRE 为可选项，实际录取中无 GRE 案例已有成功先例（如 GPA 3.41、有两段实习经历无 GRE，2023 届）（[社区来源 OpenCS](https://github.com/opencsapp/opencsapp.github.io/blob/master/docs/CE%20MEng%40TAMU.md)）
- 通常 3 月中旬后陆续出录取结果（社区来源 OpenCS）
- 国际生（尤其大陆中国学生）录取比例相对偏低，竞争度中等（社区来源 OpenCS）
- TA 岗位对英语口语要求较高，竞争激烈（社区来源 OpenCS）

## 就业情况

- 项目为 STEM 认证，支持 OPT 延期至 3 年，有利于美国求职（[官网](https://engineering.tamu.edu/cse/academics/degrees/graduate/meng.html)）
- 程序偏向 SDE 就业方向，具备研究机会但定位为专业型学位（社区来源 OpenCS）
- College Station 地处德克萨斯，周边 Houston 科技圈及达拉斯有一定就业机会；与 CMU/UW 等顶项目相比，知名度处于中等偏下（社区来源 OpenCS）
- Systems 方向师资有限（截至 Spring 2024），选课需提前规划（社区来源 OpenCS）

## 网申注意事项

- 申请通过 [EngineeringCAS](https://engineering.tamu.edu/cse/admissions-and-aid/graduate-admissions/index.html) 系统提交，非 ApplyTexas 或学校通用系统
- 所有申请人均自动纳入奖学金考虑，无需单独申请（[CSE 研究生招生页](https://engineering.tamu.edu/cse/admissions-and-aid/graduate-admissions/index.html)）
- CE MEng 与 MCS 可在入学后内部转换，可根据课程需求灵活调整（社区来源 OpenCS）
- 建议在申请时明确 CE 背景优势，因为该项目 ECEN 必修要求对 CE/EE 背景更友好（社区来源 OpenCS）
