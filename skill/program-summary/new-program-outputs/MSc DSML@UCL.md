## 课程与项目特色

项目由 UCL Computer Science 与 UCL Statistical Science 联合开设，涵盖机器学习、深度学习、贝叶斯方法、强化学习、计算机视觉等方向，亦包含数字金融、概率建模等交叉选修。项目已获 Royal Statistical Society 认证。

**必修课（3 门）**：
- Introduction to Statistical Data Science
- Applied Machine Learning
- MSc Data Science and Machine Learning Project（暑期项目，可在业界或学术机构完成）

**选修课（10+ 门，举例）**：Supervised Learning、Reinforcement Learning、Machine Vision、Bayesian Deep Learning、Applied Deep Learning 等

学生需修满 180 学分。每周约 12–16 课时接触课 + 20–25 小时自学。（[官网](https://www.ucl.ac.uk/prospective-students/graduate/taught-degrees/data-science-and-machine-learning-msc)）
