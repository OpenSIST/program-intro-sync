## 课程与项目特色

项目聚焦于计算机体系结构与底层系统设计，提供从硬件到软件的全栈嵌入式系统视角。课程设置包含四个专业化方向（[官网](https://www.tudelft.nl/en/education/programmes/masters/cese/msc-computer-embedded-systems-engineering)）：

1. **Computer Architecture（计算机体系结构）**
2. **Software（软件）**
3. **Networking（网络）**
4. **Control（控制）**

学生可选择宽口径跨方向培养，也可深入聚焦某一专业方向。课程结构为每学年四个学期（Q1–Q4），考核频率较高，学术强度大。第二学年 Q1 含三门选修，Q2–Q4 另有选修课，要求至少 3 门来自 CESE 专业核心课池。必须完成实习及毕业论文方可毕业。（[GlobalCS](https://github.com/Global-CS-application/global-cs-application.github.io/blob/main/docs/Program/T1/TUD%20MSCESE.md)）
