## 课程与项目特色

本项目由原 Computer Engineering（CE）与 Embedded Systems（ES）两个独立项目合并而来，课程体系涵盖：

- **核心课程**：嵌入式系统、计算机工程、系统工程、计算机体系结构、实时操作系统、处理器设计
- **专业化方向（Specialization Tracks）**（四选一）：
  1. Computer Architecture（计算机体系结构）
  2. Software（软件）
  3. Networking（网络）
  4. Control（控制）
- **项目结构**：核心课程 + Integration Sets（综合模块）+ 专业选修课 + 毕业论文（Thesis）
- **强制实习**：是（[社区 GlobalCS](https://github.com/Global-CS-application/global-cs-application.github.io/blob/main/docs/Program/T1/TUD%20MSCESE.md)）
- **毕业论文**：是，为项目必要组成部分

与前身 CE 项目相比，MSCESE 合并后课程选择灵活性有所下降（[社区 GlobalCS](https://github.com/Global-CS-application/global-cs-application.github.io/blob/main/docs/Program/T1/TUD%20MSCESE.md)）。
