## 学费与奖学金

- **学费**：约 $28,750/学期（2024–2025学年），3学期约合 $84,550–$86,250，另加活动费、医保等杂费（约 $3,300+/学期）。CMU SCS专业硕士项目**不提供项目自带奖学金**；部分学生可在第2–3学期担任Teaching Assistant以获得部分学费减免。（[官网学费页](https://mcds.cs.cmu.edu/learn-us-fees-and-financial-aid)）
- **奖学金**：项目本身不直接提供资助，学生可申请外部奖学金、美国联邦贷款（仅限美国公民），或寻找校内TA机会。（[官网](https://mcds.cs.cmu.edu/learn-us-fees-and-financial-aid)）

## 申请要求

- **截止日期**：Early Deadline：2025年11月19日（下午3点EST，申请费$80）；Final Deadline：2025年12月10日（下午3点EST，申请费$100）（⚠️ 以上为Fall 2026入学截止日期，请以最新官网为准）（[官网申请页](https://mcds.cs.cmu.edu/apply-mcds-program)）
- **TOEFL / IELTS**：TOEFL最低100 iBT；IELTS最低7.5；或Duolingo最低120分。英语非母语的国际学生必须提交，无豁免。（[官网](https://mcds.cs.cmu.edu/apply-mcds-program)）
- **GRE**：必须提交（CMU在读或毕业生可豁免）；Institution Code: 2074，Department Code: 0402；成绩须在5年内有效。（[官网](https://mcds.cs.cmu.edu/apply-mcds-program)）
- **推荐信**：3封（仅接受电子提交；至少2封来自教授或近期雇主）（[官网](https://mcds.cs.cmu.edu/apply-mcds-program)）

## 课程与项目特色

MCDS隶属于CMU **Language Technologies Institute（LTI）**，是SCS旗下以就业为导向的旗舰硕士项目之一。所有学生须完成至少144学分，课程结构如下（[官网课程页](https://mcds.cs.cmu.edu/learn-us-curriculum)）：

- **5门核心课程**（63学分）：Foundations of Computational Data Science、Cloud Computing（15-619）、Machine Learning（10-601）、Interactive Data Science、Data Science Seminar
- **3门方向课程**（36学分）：需在第一学期末选定以下三个方向之一：
  - **Systems**：分布式系统与大规模基础设施
  - **Analytics**：机器学习与数据分析
  - **Human-Centered Data Science（HCI）**：用户体验与社会数据应用
- **3门Capstone课程**（36学分）：以小组形式完成CMU内部研究项目或企业赞助项目
- **1门选修课**（12学分）：可从SCS 600+课程目录中自由选择

夏季实习为必选环节（不收费）。建议入学前完成在线预习课程 11-637 Foundations of Computational Data Science（需获B-以上）。（[官网](https://www.lti.cs.cmu.edu/academics/masters-programs/mcds.html)）

## 就业情况

MCDS毕业生主要进入大型科技公司担任软件工程师、数据科学家及项目经理职位。据社区记录，毕业生去向包括：OpenAI、Databricks、Snowflake、Microsoft、Apple、Qualcomm，以及各顶尖量化金融机构。官网称"MCDS graduates are sought-after software engineers, data scientists and project managers at leading information technology, software services and social media companies."（[官网](https://www.lti.cs.cmu.edu/academics/masters-programs/mcds.html)；[CSGrad来源](https://github.com/csms-apply/csgrad/blob/main/docs/S/cmu%20mcds.md)）

就业成功度与申请人"有竞争力的垂直领域经验"高度相关，而非单纯靠平台资源。（[CSGrad来源](https://github.com/csms-apply/csgrad/blob/main/docs/S/cmu%20mcds.md)）
