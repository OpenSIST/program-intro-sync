## 课程与项目特色

本项目为180学分授课型硕士，分三学期进行（[官网](https://www.ucl.ac.uk/prospective-students/graduate/taught-degrees/computer-science-msc)）：

**必修课程（6门）：**
- Introductory Programming（入门编程）
- Computer Architecture and Operating Systems（计算机体系结构与操作系统）
- Algorithmics（算法学）
- Database Fundamentals（数据库基础）
- App Engineering（应用工程）
- MSc Computer Science Project（毕业项目/论文）

**选修课程（部分示例）：**
- Functional Programming（函数式编程）
- Interaction Design（交互设计）
- Software Engineering（软件工程）
- Machine Learning（机器学习）
- Entrepreneurship（创业）

**学习强度**：每周约12-14小时课堂接触时间，加约23-26小时自主学习，合计约35-40小时/周（[官网](https://www.ucl.ac.uk/prospective-students/graduate/taught-degrees/computer-science-msc)）

**设备要求**：需自备现代电脑（最低8GB RAM，推荐16GB，500GB SSD）；机器学习方向建议配备GPU（[官网](https://www.ucl.ac.uk/prospective-students/graduate/taught-degrees/computer-science-msc)）
