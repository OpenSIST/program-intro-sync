## 课程与项目特色

项目共 **30 学分**，其中 21 学分为必修核心课、9 学分为选修课。（[官网课程页](https://datascience.columbia.edu/faq-curriculum)）

**核心课程（21 学分）：**
- **计算机科学方向（9 学分）**：COMS W4121 Computer Systems for Data Science、COMS W4721 Machine Learning for Data Science、CSOR W4246 Algorithms for Data Science
- **统计方向（9 学分）**：STAT GR5701 Probability and Statistics for Data Science、STAT GR5702 Exploratory Data Analysis and Visualization、STAT GR5703 Statistical Inference and Modeling
- **工程方向（3 学分）**：ENGI E4800 Data Science Capstone and Ethics（必修 Capstone）

**选修课（至少 9 学分）**，可跨 CS、Statistics、IEOR、商学院、公共卫生、经济学等系选修，需 4000 级及以上，例如：自然语言处理、应用深度学习、因果推断、计算机视觉、金融机器学习等。

**Capstone 项目**：所有学生须完成与业界合作伙伴（如谷歌、麦肯锡、摩根大通等）合作的 Capstone 课题，作为项目毕业核心体验。

本项目由哥伦比亚大学工学院（SEAS）与文理研究生院（GSAS）联合提供，学生可跨院系选课。
