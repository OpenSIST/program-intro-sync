## 课程与项目特色

- **核心必修（8 学分）**：CSYE 6200 面向对象设计概念、INFO 6205 程序结构与算法（[课程目录](https://catalog.northeastern.edu/graduate/engineering/multidisciplinary/software-engineering-systems-ms/)）
- **方向选择**：Coursework-only、Project（含 4 学分 Master's Project）、Thesis（含 Master's Project + 4 学分 Thesis）三条路径可选（[课程目录](https://catalog.northeastern.edu/graduate/engineering/multidisciplinary/software-engineering-systems-ms/)）
- **选修方向**：数据管理与高级数据管理、商业智能、列式数据库、大数据工程、机器学习算法、云计算、函数式编程（Scala）、多线程并发计算（[官网](https://coe.northeastern.edu/academics-experiential-learning/academic-departments/mgen/ms-swes/)）
- **Gordon 工程领导力证书**：可与本项目并修，最多 16 学分可计入选修要求（[课程目录](https://catalog.northeastern.edu/graduate/engineering/multidisciplinary/software-engineering-systems-ms/)）
- **可选 Co-op**：最长 8 个月，学校拥有逾 3,000 家雇主合作伙伴（[官网](https://coe.northeastern.edu/academics-experiential-learning/academic-departments/mgen/ms-swes/)）
- **入学时间**：Fall 和 Spring 均可申请；支持全职与兼职（[官网](https://graduate.northeastern.edu/programs/ms-software-engineerng-systems/master-of-science-in-software-engineering-systems/)）
