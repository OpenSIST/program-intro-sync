## 学费与奖学金

- **学费**：约 USD 41,994 / 年（36 学分总计，以当年 GSAS 官方公布为准）（[GSAS Tuition Page](https://cds.nyu.edu/admissions/)）
- **奖学金**：提供数量有限的竞争性学费奖学金，录取审核中自动考量，无需单独申请，奖学金可覆盖部分学费最长两年（[官网](https://cds.nyu.edu/masters-in-data-science-faq/)）

## 申请要求

- **截止日期**：每年 **2 月 14 日** 17:00 EST（仅秋季入学，无春季）（[官网](https://cds.nyu.edu/admissions/)，⚠️ 请以最新官网为准）
- **TOEFL / IELTS**：母语非英语且未在英语授课大学取得学位者需提交；**TOEFL 最低推荐 100 分**（网考），**IELTS 最低推荐 7.0**（[官网](https://cds.nyu.edu/masters-in-data-science-faq/)）
- **GRE**：可选，接受非官方成绩上传；**不接受 GMAT**（[官网](https://cds.nyu.edu/masters-in-data-science-faq/)）
- **推荐信**：3 封（建议提供教授或雇主推荐，最好有抬头信纸）（[官网](https://cds.nyu.edu/admissions/masters-in-data-science-admissions-req/admissions-requirements/)）

### 前置课程要求

以下课程须在申请前完成（[官网](https://cds.nyu.edu/admissions/masters-in-data-science-admissions-req/admissions-requirements/)）：
1. Calculus I
2. Linear Algebra
3. Intro to Computer Science（需有 Python 或 R 基础）
4. 以下之一：Calculus II、Probability、Statistics 或同等水平高阶定量课程

加分项：有机器学习、计算统计学或数据挖掘经验

> ⚠️ 注意：NYU 现有研究生不允许内部转项，需重新申请。

## 课程与项目特色

项目共 36 学分，提供以下三个方向（[官网](https://cds.nyu.edu/masters-in-data-science-overview/)）：

1. **Industry Concentration**：结合行业课程与强制实习，偏职业导向
2. **Biomedical Informatics Track**：与生物医学导师合作完成 Capstone 项目
3. **Data Science Track**（三个兴趣方向）：
   - Big Data
   - Mathematics and Data
   - Natural Language Processing

所有方向均包含 Capstone 项目；NYU CDS 与行业合作方合作为学生提供职业发展活动及校内 RA/TA 机会（竞争性，不保证）。
