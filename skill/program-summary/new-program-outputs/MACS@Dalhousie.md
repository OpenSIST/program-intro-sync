## 课程与项目特色

项目为期 16 个月，分三个学术学期（每学期修 3 门课）加一个工作学期（[官网课程大纲](https://www.dal.ca/study/program-sites/applied-computer-science/program-overview/program-outline.html)）：

**核心必修课**（所有学生）：
- CSCI5100、CSCI5308、CSCI5408
- 实习准备课 CSCI9890

**选修方向**：
- Internship 轨道：CSCI9100（行业实习）
- Project 轨道：CSCI9301 / CSCI9302（科研项目）
- 最多选修 2 门 6000 级课程

**证书选修**（可选）：
- Certificate in Web and Mobile Computing
- Certificate in Cloud Data Analytics

**特色课程领域**：移动计算、数据管理、数据仓库与分析、通信系统等。Entrepreneurship 轨道学生须额外修完 2 门商学院研究生课（BUSI）。

## 就业情况

实习典型薪资约 **CA$28.32/小时**；典型岗位包括软件开发工程师、解决方案架构师、技术顾问、QA 分析师、移动开发工程师、数据仓库架构师等；主要行业涵盖软件开发、电信、医疗、政府及电子商务（[官网 WIL 页面](https://www.dal.ca/faculty/wil/disciplines/it/masters_applied_computer_science.html)）。

由于该项目为 Dalhousie 较新的研究生项目，历史就业数据尚有限，建议向项目组索取最新就业统计（联系 graduate@cs.dal.ca）。
