## 课程与项目特色

项目共需修满 **39 学分，13 门课**，其中约半数来自 NYU Courant 计算机科学，半数来自 Stern 商学院 MBA 课程，选课优先级仅次于 Stern MBA 学生。（[官网](https://cs.nyu.edu/home/master/prospective_msis.html) / [社区](https://github.com/opencsapp/opencsapp.github.io/blob/master/docs/MSIS%40NYU%20Courant.md)）

**核心 CS 课程（官网示例）**：
- Fundamental Algorithms（必修）
- Data Communications & Networks / Operating Systems / Database Systems（选 2 门）
- Computer Science 选修（含密码学、自然语言处理、分布式计算等）

**Stern 商科课程**：
- 领导力、市场营销、财务、运营管理、数据科学与商业分析、Web/移动应用设计

**实践课**：Information Technology Projects（必修，与本地企业/非营利/政府机构合作，团队解决真实 IT 问题）

**特色**：可在项目中途申请转换为 MSCS 赛道（[官网](https://cs.nyu.edu/home/master/prospective_msis.html) / [社区](https://github.com/opencsapp/opencsapp.github.io/blob/master/docs/MSIS%40NYU%20Courant.md)）

（[NYU Bulletins](https://bulletins.nyu.edu/graduate/arts-science/programs/information-systems-ms/)）

## 就业情况

毕业生常见去向（[官网](https://cs.nyu.edu/home/master/prospective_msis.html)）：
- **职位**：Product Manager、Systems Architect、CIO、Strategic Technology Consultant、IT 团队负责人
- **行业**：软件厂商、企业 IT 部门、咨询公司
- 社区帖提及通过 Career Fair 和 Networking 获得 offer，Courant 与科技公司有较强行业联系（[一亩三分地 2019Fall 总结](https://www.1point3acres.com/bbs/thread-638807-1-1.html)）

## 网申注意事项

- 申请系统为 GSAS 统一入口：[apply.gsas.nyu.edu](https://apply.gsas.nyu.edu/apply/)，申请费 **$110**（[官网](https://gsas.nyu.edu/admissions/arc/programs/computer-science.html)）
- 建议提交可选的 **Video Statement**，MSIS 官网特别提及此项（[GSAS 招生页](https://gsas.nyu.edu/admissions/arc/programs/computer-science.html)）
- **NYU 不为 MSIS 学生提供任何形式的经济援助**，请提前规划资金（[官网](https://cs.nyu.edu/home/master/prospective_msis.html)）
- 项目可从 MSIS 转 MSCS，有意向的同学可在入学后根据课业表现申请转换（[官网](https://cs.nyu.edu/home/master/prospective_msis.html) / 社区）
- 部分课程（如 Operating Systems）工作量极大（单次 lab 可达 20+ 小时），申请前需做好心理准备（[一亩三分地](https://www.1point3acres.com/bbs/thread-638807-1-1.html)）
