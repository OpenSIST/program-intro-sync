## 课程与项目特色

项目为纯课程制，共需完成 30 学分，核心结构如下（[官网](https://iac.vt.edu/masters-degrees/meng-cs.html)）：

- **必修**：CS5024（职业道德）、CS5934（Capstone 设计项目）、CS5040 或同等数据结构课程
- **集中方向**：需在所选方向内完成至少 3 门课（软件开发与应用 / 数据分析与 AI / 计算机安全 / 人机交互）
- **项目轨道**：
  - *Project Enhanced Track*：选课灵活度较高，通过项目强化 CS 实际应用
  - *Project Intensive Track*：强调深度项目体验与结构化开发实践
- NCR 校区采用**晚间课程**模式（每周上课一次），兼顾在职学生需求（[NCR 官网](https://www.eng.ncr.vt.edu/CS/index.html)）
