## 课程与项目特色

项目分为两大方向（[官网](https://programmes.polytechnique.edu/en/master/all-msct-programs/artificial-intelligence-and-advanced-visual-computing) / [项目 Wiki](https://msct.dix.polytechnique.fr/aivic/wiki/)）：

- **人工智能方向**：统计学习、数据分析、深度学习
- **视觉计算方向**：3D 计算机图形学、虚拟现实与增强现实、多模态交互、计算机视觉、机器人学、3D 制造

学年结构：
- 第1–2学期（9月至次年3月）：课程 + 实践课 + 研讨会
- 第3学期（4月至9月）：6个月实习（企业研发中心或研究实验室）
- 每年第一周：数学与计算机科学基础补习课

项目特色：
- 3D 视觉与计算机图形学为顶尖优势方向
- 毕业后该项目更名为 Visual and Creative AI（ViCAI），2026 年 Eduniversal 排名该方向全国第一
- 有毕业论文要求
- 与萨克雷大学城研究机构紧密合作
- 工业合作伙伴提供横向项目机会
