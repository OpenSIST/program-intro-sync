- **截止日期**：Fall 入学优先截止 **12月1日**，最终截止 **2月15日**（历史数据延期至 3月15日）；Spring 入学截止 **11月1日**（[官网](https://engineering.nyu.edu/admissions/graduate/apply/application-deadlines)，⚠️ 请以最新官网为准）

## 课程与项目特色

项目共需 **30 学分**，课程结构如下（[NYU Bulletins](https://bulletins.nyu.edu/graduate/engineering/programs/computer-engineering-ms/)）：

- **核心课程（6 学分）**：从以下 5 门中选 2 门：
  - Internet Architecture & Protocols (ECE-GY 6353)
  - Advanced Hardware Design (ECE-GY 6463)
  - Introduction to VLSI System Design (ECE-GY 6473)
  - Real Time Embedded Systems (ECE-GY 6483)
  - Computing Systems Architecture (ECE-GY 6913)
- **ECE-GY 选修课（18 学分）**
- **自由选修（3 学分）**：可选 ECE-GY 或其他系课程
- **Advanced Project / Thesis（3 学分）**：可选高级项目 (ECE-GY 9953) 或硕士论文 (ECE-GY 997X) 或 CS 高级项目 (CS-GY 9963)

项目设有三个方向（[官网](https://engineering.nyu.edu/academics/programs/computer-engineering-ms)）：
1. **人工智能（AI）**：机器学习、深度学习、随机过程
2. **软件（Software）**：嵌入式系统、VLSI 设计、网络与计算架构
3. **机器人（Robotics）**：机器人导航、控制系统、自主系统

**特点**：课程选择灵活度较高，学生可以**全选软件课程而无需修硬件课**，对 EE 背景及转码学生友好；配备 NYU Tandon Bridge Program，供缺乏 CE 相关背景的学生补习（[社区](https://github.com/opencsapp/opencsapp.github.io/blob/master/docs/CE%40NYU%20Tandon.md)）。
